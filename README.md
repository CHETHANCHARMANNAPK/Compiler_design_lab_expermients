# Compiler Design Lab Assignments

Lab experiments for Compiler Design course, implemented in Python.

## Experiments

| # | Topic | File |
|---|-------|------|
| 1 | Lexical Analysis | experiment-1/lexical_analyzer.py |
| 2 | RE to NFA (Thompson's Construction) | experiment-2/re_to_nfa.py |
| 3 | NFA to DFA (Subset Construction) | experiment-3/nfa_to_dfa.py |
| 4 | Left Recursion & Left Factoring | experiment-4/eliminate_ambiguity.py |
| 5 | FIRST and FOLLOW Sets | experiment-5/first_follow.py |
| 6 | Predictive Parsing Table (LL1) | experiment-6/predictive_parser.py |
| 7 | Shift-Reduce Parsing | experiment-7/shift_reduce_parser.py |
| 8 | LEADING and TRAILING Sets | experiment-8/leading_trailing.py |

## Running

```
python experiment-1/lexical_analyzer.py
```

Each program runs built-in examples first, then allows custom input.
